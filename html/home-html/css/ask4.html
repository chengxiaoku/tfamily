关于如何提问

先搜索，再提问

你提问前有在我们的 搜索栏 先行搜索过吗？使用搜索（引擎），能更快地帮你找到答案。即使没找到，在看了相关或者类似的问题之后，你的提问会更准确。

尽量清楚、详细地描述问题

  • √标题 清晰明了，内容 包含必要的操作环境、截图和代码、期望结果与实际结果
  • √善用 编辑器 排版你的问题，提高可读性
  • √准确地使用多个 标签 标记你的问题
保持求知欲

或许最后得到的答案并不是你最想要的，但深思熟虑过的问题依旧可能会让你有其他方面的收获。每个人的成长都是一步步来的，所以，Keep an open mind。
